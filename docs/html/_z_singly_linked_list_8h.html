Affiche les données de la liste selon une forme définie par un format.

dataPerLineLe nombre de données à afficher par ligne, mettre 0 si l'on ne veut pas de mise à la ligne automatique
formatLe format pour afficher correctement les données :

  • "d" -> int32_t

  • "o" -> octal
  • "x" -> hexadécimal
  • "c" -> char
  • "s" -> char*
  • "f" -> float
  • "lf" -> double
