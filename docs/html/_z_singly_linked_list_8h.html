Affiche les données de la liste selon une forme définie par un format.

Permet d'afficher un nombre de données par ligne de la liste dans un certain format

Parameters
listUn pointeur vers la liste
dataPerLineLe nombre de données à afficher par ligne
formatLe format pour afficher correctement les données :
  • "p" -> pointeur
  • "d" -> int32_t
  • "u" -> uint32_t
  • "o" -> octal
  • "x" -> hexadécimal
  • "c" -> char
  • "s" -> char*
  • "f" -> float
  • "lf" -> double
